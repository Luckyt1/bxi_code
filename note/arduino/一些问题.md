-  `unbutu`中容易出现权限问题
```
sudo nano /etc/udev/rules.d/49-stm32dfu.rules
#创建规则文件

#写入文件
# STM32 DFU mode
SUBSYSTEM=="usb", ATTR{idVendor}=="0483", ATTR{idProduct}=="df11", MODE="0666"
# Arduino DFU mode
SUBSYSTEM=="usb", ATTR{idVendor}=="2341", ATTR{idProduct}=="0070", MODE="0666"
SUBSYSTEM=="usb", ATTR{idVendor}=="2341", ATTR{idProduct}=="0037", MODE="0666"

#运用规则
sudo udevadm control --reload-rules
sudo udevadm trigger
```
